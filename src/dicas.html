Controles Básicos Para controlar sua nave, use as seguintes teclas no teclado:
Seta para cima: Move a nave para cima
Seta para baixo: Move a nave para baixo
Seta para esquerda: Move a nave para a esquerda
Seta para direita : Move a nave para a direita
Barra de Espaço (Space): Dispara as armas da nave


Movimentação: Use as teclas W, S, A, D para navegar sua nave pelo espaço.
Evite os obstáculos e os tiros inimigos movendo-se rapidamente em diferentes direções.

Ataque: Pressione a tecla Barra de Espaço para disparar suas armas.
Mire cuidadosamente nos inimigos e nos pontos fracos de Malígioth para causar o máximo de dano.
